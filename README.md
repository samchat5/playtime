# PlayTime

## Requirements

First things first you need Node. To see instructions for installing it on your OS of choice, click [here](https://nodejs.org/en/download/)

## Installation

    $ git clone https://github.com/samchat5/playtime.git/
    $ npm install

### Starting

    $ npm start
